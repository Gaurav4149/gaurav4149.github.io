Background threads continuously poll CPU%, memory, disk I/O and active process list using psutil — without ever blocking the UI thread for a lag-free experience.

Data streams in real-time into embedded Matplotlib charts inside Tkinter, with configurable refresh intervals and smooth animated curve rendering.

Browse all running processes, inspect resource usage, and terminate any process by PID directly from the GUI — full system control without a terminal.

HOW IT WORKS — STEP BY STEP

01

PACKET CAPTURE

Uses Scapy to capture raw network packets on a selected interface in real-time — supporting TCP, UDP, ICMP, ARP, and DNS traffic types.

02

LAYER-BY-LAYER DISSECTION

Each packet is parsed Ethernet → IP → Transport → Application, extracting source/dest IPs, ports, protocol flags, TTL values, and payload summaries.

03

ANOMALY DETECTION

Flags suspicious patterns: SYN floods, ARP spoofing attempts, rapid port scanning, and abnormal data volumes — displayed with timestamp and severity level.
